# ROADMAP.md
## Agentic HTML Protocol
### Experimental Research Roadmap

---

# Introducción

Este roadmap describe la dirección experimental prevista para Agentic HTML Protocol (AHP).

El objetivo del roadmap no es prometer funcionalidades definitivas.

El objetivo es documentar:
- líneas de investigación
- hipótesis futuras
- infraestructura experimental
- evolución conceptual del protocolo

AHP sigue siendo un proyecto experimental.

Todas las versiones y direcciones futuras están sujetas a cambios.

---

# Filosofía del Roadmap

AHP no evoluciona únicamente como:
- especificación HTML
- sistema de metadata
- framework frontend

AHP evoluciona como:
- investigación de semántica operacional
- navegación agentic
- comportamiento operacional autónomo
- interacción humano-agente

AHP actualmente aún tiene limitaciones importantes:

- poca instrumentación

- pocas interfaces benchmarkeadas
- observación parcialmente manual
- falta de telemetry

# Objetivos de Investigación a Largo Plazo

AHP investiga si es posible:

- reducir dependencia heurística
- aumentar navegación determinista
- reducir ambigüedad operacional
- mejorar continuidad contextual
- mejorar targeting semántico
- reducir acciones peligrosas
- mejorar coordinación humano-agente

---

# No Objetivos

AHP NO busca:
- reemplazar HTML
- reemplazar APIs
- eliminar interacción humana
- automatizar completamente la web
- crear una web exclusiva para agentes
- imponer un estándar obligatorio

---

# Visión General

La visión de AHP es explorar si la web puede evolucionar desde:

```txt
interfaces visuales interpretadas heurísticamente
```

hacia:

```txt
interfaces operacionalmente interpretables
```

para agentes inteligentes autónomos.

---

# Riesgos Reconocidos

Existen riesgos importantes:

- complejidad semántica excesiva
- sobreinstrumentación
- falsas correlaciones
- benchmark overfitting
- degradación UX humana
- semántica inconsistente
- dependencia de implementación

Estos riesgos forman parte de la investigación.

---

# Estado Científico

AHP actualmente:
- no es estándar oficial
- no es especificación W3C
- no es framework consolidado
- no es tecnología validada académicamente

AHP es:
- investigación experimental temprana

---

# Principio Humano

La prioridad sigue siendo:
- claridad humana
- accesibilidad
- usabilidad
- interfaces centradas en personas

La semántica operacional debe complementar la experiencia humana, no reemplazarla.

---

# Declaración Final

AHP investiga una pregunta central:

> ¿Puede la web exponer semántica operacional suficiente para modificar comportamiento de agentes autónomos durante interacción real?

La respuesta todavía es experimental.
