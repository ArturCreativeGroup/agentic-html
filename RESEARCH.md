# RESEARCH.md
## Agentic HTML Protocol v0.2
### Research Notes & Experimental Foundations

---

# Introducción

Agentic HTML Protocol (AHP) es una línea experimental de investigación orientada a estudiar si una capa semántica operacional integrada en interfaces web puede modificar el comportamiento de agentes autónomos durante ejecución de tareas reales.

El proyecto nace de una observación fundamental:

> La web moderna fue diseñada principalmente para humanos, no para agentes autónomos.

Aunque modelos modernos pueden interpretar interfaces visualmente, gran parte de la interacción agentic actual depende de:
- exploración heurística
- reinterpretación constante
- targeting ambiguo
- navegación reactiva
- inferencia contextual improvisada

AHP investiga si semántica operacional explícita puede reducir esa dependencia heurística.

---

# Hipótesis Principal

La hipótesis central de AHP es:

> Una capa semántica operacional puede modificar patrones de comportamiento operacional de agentes autónomos durante navegación e interacción web.

Específicamente, AHP explora si esta semántica puede:

- reducir exploración heurística
- aumentar navegación determinista
- mejorar continuidad operacional
- reducir pausas cognitivas
- mejorar targeting contextual
- reducir reinterpretación innecesaria
- disminuir acciones peligrosas

---

# Contexto del Problema

La mayoría de interfaces web actuales:
- están optimizadas visualmente para humanos
- no exponen intención operacional explícita
- no describen flujo de interacción
- no describen restricciones operacionales
- no describen políticas de seguridad
- no exponen continuidad contextual para agentes

Como resultado:
- los agentes deben improvisar navegación
- interpretar señales ambiguas
- explorar visualmente
- reconstruir intención continuamente

Esto genera:
- ineficiencia operacional
- errores
- rescaneos
- targeting inconsistente
- loops de navegación
- pausas cognitivas

---

# Filosofía de AHP

AHP no intenta:
- reemplazar HTML
- reemplazar accesibilidad
- reemplazar APIs
- crear una web paralela
- eliminar humanos del flujo

AHP propone:

> Human First, Agent Ready

La interfaz sigue siendo humana, pero expone contexto operacional adicional para agentes inteligentes.

# Evolución Conceptual

---

# v0.1
## Structural Semantic Layer

La primera versión experimental introdujo:
- metadata inline
- agentic.json
- agentic-map.json
- llms.txt
- atributos semánticos estructurales

Objetivo:
mejorar comprensión contextual básica.

Resultados observados:
- menor ambigüedad estructural
- navegación más clara
- reducción inicial de exploración heurística

---

# v0.2
## Operational Semantic Layer

v0.2 expande el protocolo desde:
- estructura

hacia:
- comportamiento operacional

Introduciendo:
- AHP-SKILL.md
- dangerous action policies
- grouped execution
- reanalysis policies
- deterministic interaction guidance
- benchmark framework

---

# Cambio Conceptual Principal

La transición importante de v0.2 es:

v0.1:
> “qué es la interfaz”

v0.2:
> “cómo debe ejecutarse operacionalmente”

Esto convierte AHP en:
- una capa semántica operacional
y no únicamente:
- metadata estructural

# Resultados Iniciales Observados

---

# ObraLink Test

Comparación:
- baseline
- vs AHP v0.1

Resultados observados:
- ~18.2% reducción estimada de tiempo
- ~20% reducción estimada de pasos

Cambios observados:
- menor exploración heurística
- mayor planificación previa
- navegación más semántica
- menor incertidumbre operacional

---

# Qwen Desktop Test

Resultados observados:
- ~37.1% reducción estimada de pasos

Cambios observados:
- menor ambigüedad
- targeting más claro
- navegación más directa

---

# ObraLink AHP v0.2

Resultados observados:
- 100% completion rate
- 0 errores fatales
- movement batching
- reducción de pausas cognitivas
- avoidance de Enter
- reanálisis localizado
- continuidad operacional mantenida

---

# Observación Más Importante

El hallazgo más relevante no parece ser:
- velocidad

Sino:
- cambio de comportamiento operacional

Cambios observados:
- mayor planificación
- menor improvisación
- menos reinterpretación global
- targeting más preciso
- navegación más lineal
- menor dependencia heurística

# Interpretación Prudente

Los resultados actuales NO son:
- concluyentes
- estadísticamente validados
- generalizables universalmente

Las pruebas actuales:
- tienen pocas muestras
- dependen parcialmente de observación manual
- usan pocas interfaces
